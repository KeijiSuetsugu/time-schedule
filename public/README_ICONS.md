# PWAアイコン画像について

このディレクトリには以下のアイコン画像を配置してください：

- `icon-192x192.png` - 192x192ピクセルのPNG画像
- `icon-512x512.png` - 512x512ピクセルのPNG画像
- `favicon.ico` - ファビコン

これらの画像がない場合でもアプリケーションは動作しますが、PWAとしてインストールした際のアイコンが表示されません。

## アイコンの作成方法

1. オンラインツールを使用（例: https://realfavicongenerator.net/）
2. デザインツール（Figma、Photoshopなど）で作成
3. 簡単なロゴやテキストから生成

最低限、`icon-192x192.png` と `icon-512x512.png` を作成することを推奨します。
